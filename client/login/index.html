Tech Stack

Reactjs / React Native

One way data flow creates a very fast user experience and our app takes advantage of Reacts virtual DOM to render incremental changes.

Nodejs / Express

Obie utilizes a RESTful API as well as express middleware to ensure security with JWTs and express sessions.

MySql

The SQL database delivers rapid data retrieval for the relational data associated with the application.

Socket.io

Obie utilizes sockets to ensure a posivite user experience. All data is live and reloaded as soon as another user posts it.
